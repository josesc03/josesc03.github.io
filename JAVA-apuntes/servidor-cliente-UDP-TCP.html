Guía de Métodos de Sockets

Ejemplo de uso combinado

Este código muestra cómo usar varias funciones de sockets en un cliente-servidor.

// Servidor
ServerSocket server = new ServerSocket(8080, 50);
System.out.println("Servidor iniciado en el puerto " + server.getLocalPort());
Socket clientSocket = server.accept();
System.out.println("Cliente conectado desde el puerto " + clientSocket.getPort());

// Cliente
InetAddress address = InetAddress.getByName("localhost");
Socket socket = new Socket(address, 8080);
System.out.println("Conectado al servidor");

// Comunicación - Cliente
DataOutputStream output = new DataOutputStream(socket.getOutputStream());
output.writeUTF("Hola desde el cliente");

// Comunicación - Servidor
DataInputStream input = new DataInputStream(clientSocket.getInputStream());
System.out.println("Mensaje recibido: " + input.readUTF());

Conexión Peer-to-Peer con Datagramas

Se utiliza la clase DatagramSocket para enviar y recibir paquetes de datos.

// Envío de Datagramas
DatagramSocket sender = new DatagramSocket();
byte[] buffer = "Hola, peer".getBytes();
InetAddress address = InetAddress.getByName("localhost");
DatagramPacket packet = new DatagramPacket(buffer, buffer.length, address, 8080);
sender.send(packet);
System.out.println("Mensaje enviado");

// Recepción de Datagramas
DatagramSocket receiver = new DatagramSocket(8080);
byte[] receiveBuffer = new byte[1024];
DatagramPacket receivedPacket = new DatagramPacket(receiveBuffer, receiveBuffer.length);
receiver.receive(receivedPacket);
String message = new String(receivedPacket.getData(), 0, receivedPacket.getLength());
System.out.println("Mensaje recibido: " + message);
