Cheat Sheet XAML

Layouts (Contenedores)

Ejemplo de <Grid>:

<Grid>
    <Grid.RowDefinitions>
        <RowDefinition Height="Auto" />
        <RowDefinition Height="*" />
    </Grid.RowDefinitions>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="Auto" />
        <ColumnDefinition Width="*" />
    </Grid.ColumnDefinitions>

    <Label Grid.Row="0" Grid.Column="0" Text="Etiqueta en Fila 0, Columna 0"/>
    <Button Grid.Row="1" Grid.Column="1" Text="Botón en Fila 1, Columna 1"/>
</Grid>

Ejemplo de <StackLayout>:

<StackLayout Orientation="Vertical">
    <Label Text="Primera etiqueta" FontSize="24" />
    <Button Text="Haz click aquí" />
    <Image Source="imagen.png" />
</StackLayout>

Atributos Generales de XAML

Margin: Espacio exterior que define la separación entre el borde del elemento y otros elementos alrededor.

<Button Text="Botón" Margin="10,5,10,5"/>

Padding: Espacio interior que define la separación entre el contenido del elemento y su borde.

<Button Text="Botón" Padding="10"/>

Width: Define el ancho del elemento.

<Button Text="Botón" Width="200"/>

Height: Define la altura del elemento.

<Button Text="Botón" Height="50"/>

HorizontalAlignment: Define la alineación horizontal del elemento dentro de su contenedor.

<Button Text="Botón" HorizontalAlignment="Center"/>

VerticalAlignment: Define la alineación vertical del elemento dentro de su contenedor.

<Button Text="Botón" VerticalAlignment="Top"/>

Visibility: Define si un elemento es visible o no.

<Button Text="Botón" Visibility="Collapsed"/>

Opacity: Define la transparencia del elemento.

<Button Text="Botón" Opacity="0.5"/>

RenderTransform: Aplica una transformación visual a un elemento, como escalado o rotación.

<Button Text="Botón">
    <Button.RenderTransform>
        <ScaleTransform ScaleX="1.5" ScaleY="1.5"/>
    </Button.RenderTransform>
</Button>

ToolTip: Muestra un texto adicional cuando se pasa el ratón sobre el elemento.

<Button Text="Botón" ToolTip="Haz clic para realizar una acción"/>

Cursor: Define el cursor que aparece cuando el ratón está sobre el elemento.

<Button Text="Botón" Cursor="Hand"/>

Eventos Más Usados

Eventos más comunes de <Button>:

1. Clicked: Se dispara cuando se hace clic en el botón.
   <Button Text="Botón" Clicked="OnButtonClicked"/>

2. Pressed: Se dispara cuando se presiona el botón.
   <Button Text="Botón" Pressed="OnButtonPressed"/>

3. Released: Se dispara cuando se suelta el botón.
   <Button Text="Botón" Released="OnButtonReleased"/>

Cómo manejar estos eventos en C#:

// Método que se dispara al hacer clic en el botón
void OnButtonClicked(object sender, EventArgs e) {
    Console.WriteLine("Botón clicado");
}

// Método que se dispara al presionar el botón
void OnButtonPressed(object sender, EventArgs e) {
    Console.WriteLine("Botón presionado");
}

// Método que se dispara al soltar el botón
void OnButtonReleased(object sender, EventArgs e) {
    Console.WriteLine("Botón soltado");
}

Eventos más comunes de <Entry>:

1. TextChanged: Se dispara cuando el texto cambia.
   <Entry Text="Texto" TextChanged="OnTextChanged"/>

2. Completed: Se dispara cuando se completa la entrada.
   <Entry Completed="OnEntryCompleted"/>

Cómo manejar estos eventos en C#:

// Método que se dispara cuando el texto cambia
void OnTextChanged(object sender, TextChangedEventArgs e) {
    Console.WriteLine("Texto cambiado a: " + e.NewTextValue);
}

// Método que se dispara al completar la entrada
void OnEntryCompleted(object sender, EventArgs e) {
    Console.WriteLine("Entrada completada");
}

Eventos más comunes de <Picker>:

1. SelectedIndexChanged: Se dispara cuando el índice seleccionado cambia.
   <Picker SelectedIndexChanged="OnPickerSelectedIndexChanged"/>

2. SelectedItemChanged: Se dispara cuando el ítem seleccionado cambia.
   <Picker SelectedItemChanged="OnPickerSelectedItemChanged"/>

Cómo manejar estos eventos en C#:

// Método que se dispara al cambiar el índice seleccionado
void OnPickerSelectedIndexChanged(object sender, EventArgs e) {
    var picker = (Picker)sender;
    Console.WriteLine("Índice seleccionado: " + picker.SelectedIndex);
}

// Método que se dispara al cambiar el ítem seleccionado
void OnPickerSelectedItemChanged(object sender, EventArgs e) {
    var picker = (Picker)sender;
    Console.WriteLine("Ítem seleccionado: " + picker.SelectedItem);
}

Eventos más comunes de <ListView>:

1. ItemTapped: Se dispara cuando se toca un ítem en la lista.
   <ListView ItemTapped="OnListViewItemTapped"/>

2. ItemSelected: Se dispara cuando se selecciona un ítem.
   <ListView ItemSelected="OnListViewItemSelected"/>

Cómo manejar estos eventos en C#:

// Método que se dispara al tocar un ítem en la lista
void OnListViewItemTapped(object sender, ItemTappedEventArgs e) {
    if (e.Item != null) {
        Console.WriteLine("Ítem tocado: " + e.Item);
    }
}

// Método que se dispara al seleccionar un ítem
void OnListViewItemSelected(object sender, SelectedItemChangedEventArgs e) {
    if (e.SelectedItem != null) {
        Console.WriteLine("Ítem seleccionado: " + e.SelectedItem);
    }
}

Uso de x:

El prefijo x: se utiliza en XAML para hacer referencia a tipos o elementos que pertenecen al espacio de nombres de XAML. Aquí hay algunos ejemplos comunes:

  • x:Name: Asigna un nombre a un elemento para que se pueda referenciar en el código detrás. 
    <Label x:Name="lblmyLabel" Text="Hola Mundo"/>
    En el código C#, puedes acceder al Label así: 
    lblmyLabel.Text = "Nuevo Texto";
  • x:Key: Se utiliza para definir una clave única en recursos (como estilos o plantillas). 
    <ResourceDictionary>
            <Style x:Key="myButtonStyle" TargetType="Button">
                <Setter Property="BackgroundColor" Value="LightBlue"/>
            </Style>
        </ResourceDictionary>
    Luego puedes usarlo así: 
    myButton.Style = (Style)Resources["myButtonStyle"];
  • x:Static: Permite acceder a un miembro estático de una clase. 
    <Label Text="{x:Static local:MiClase.MiPropiedad}"/>
  • x:Type: Se utiliza para referirse a un tipo en el espacio de nombres de XAML. 
    <Label Text="Tipo de objeto: {x:Type local:MiTipo}"/>
